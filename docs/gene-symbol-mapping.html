3.2 Finding annotation resources

From the airway [vignette][airway-vignette], the experiment used Ensembl reference annotations from Ensembl release 75. This is available as a Bioconductor package

BiocManager::available("EnsDb")
## [1] "AHEnsDbs"              "EnsDb.Hsapiens.v75"    "EnsDb.Hsapiens.v79"   
## [4] "EnsDb.Hsapiens.v86"    "EnsDb.Mmusculus.v75"   "EnsDb.Mmusculus.v79"

so we load that package (it may be necessary to install the package using BiocManager, as outlined previously)

library(EnsDb.Hsapiens.v75)
