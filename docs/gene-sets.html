  toptbl %>%

Biologically, genes always function in some kind of relation with other genes and the biology of organsims. For instance, genes are expressed in particularly cellular compartments, or participate in particular biological processes. We would like to use established ontologies that describe these relationships.

The GO.db package provides information on GO identifiers.

goid <- "GO:0007165"

5.4 More flexible gene set annotation using BiocSet

Gene sets can be thought of as a three related ‘tables’: genes, sets, and the gene/set relationship. BiocSet provides a way to manage these tables.

library(BiocSet)

Start by identifying the unique element / set combinations

elementsets <-
  eggo %>%
  dplyr::select(element = "ENTREZID", set = "GOID")

Then create a BiocSet from this relationship

es <- BiocSet_from_elementset(elementsets)

Use left_join_*() to join annotations to the elements and sets, first removing the duplicate information in eggoinfo cause by including the ‘GOID’ column

eggoinfo1 <-
  eggoinfo %>%
  dplyr::select(-ENTREZID) %>%
  distinct()
es <-
  es %>%
  left_join_element(toptbl, by = c(element = "ENTREZID")) %>%
  left_join_set(eggoinfo1, by = c(set = "GOID"))
